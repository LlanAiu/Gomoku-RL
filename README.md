# Gomoku RL Boil

Starter / source code for our implementation of simple RL mechanics.

We're also playing Gomoku, because just building out the machinery is rather boring.

This repository is organized as a web application monorepo, so specific setup steps are in the respective `frontend/README.md` and `backend/README.md`.

You may also find it helpful, just in terms of ease of reading these MD files, to find the markdown preview in your IDE (for VS Code: right-click on .md file --> Open Preview).
